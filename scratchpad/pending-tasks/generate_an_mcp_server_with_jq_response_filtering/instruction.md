Model Context Protocol (MCP) servers generated by Speakeasy allow AI agents like Claude to securely interact with external APIs. However, large JSON API responses can easily exhaust an LLM's context window. 

You need to configure `gen.yaml` to target an MCP server and apply a JQ filter within the configuration to automatically truncate large JSON array responses down to the first 50 items.

**Constraints:**
- The primary generation target in the configuration MUST be an MCP server.
- The JQ filter MUST be defined correctly in the Speakeasy configuration to process outbound responses to the AI agent.
